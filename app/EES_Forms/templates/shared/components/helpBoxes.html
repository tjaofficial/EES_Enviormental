{% elif selector == "frequencyInput" %}
HELP: Frequency Input

This input allows you to specify the time interval at which forms should be selected for printing. Based on the selected frequency:

  • Daily: All forms due within each day will be included for that day's selection.
  • Weekly: Forms due within the week (Monday to Sunday) will be included for that week's selection.
  • Monthly: Forms that are due within the specified month will be included for that month's selection.
  • Custom Range: Select a specific date range to include all forms due within that period.

The frequency setting ensures that forms are grouped by the chosen interval, making it easier to manage and print the relevant forms for the specified time frame.
